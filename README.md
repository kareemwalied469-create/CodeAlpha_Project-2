## Project Overview

This project is a simple text-based Hangman game developed using Python.  
The main idea of the game is to let the player guess a hidden word one letter at a time. The program randomly selects a word from a predefined list, and the player must continue guessing until the word is completed or the number of allowed wrong attempts is finished.

The project is designed for beginners to practice important Python programming concepts such as:
- Random word selection using the `random` module
- Using loops (`while`)
- Decision making with `if-else`
- Working with strings and lists
- Taking user input from the console

The game runs completely in the terminal/console without graphics or audio, making it simple and easy to understand for new programmers.
